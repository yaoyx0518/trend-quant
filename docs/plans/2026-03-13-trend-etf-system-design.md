# 趋势ETF量化系统设计纪要（已确认）

- 日期：2026-03-13
- 阶段：需求澄清完成，进入PRD与开发文档阶段
- 目标：先文档定稿，再进入开发

## 1. 范围与目标

- 市场：A股场内ETF
- V1 ETF池（可配置，不写死）：
  - `510330.SS`
  - `512500.SS`
  - `512100.SS`
  - `563300.SS`
- 策略：自定义趋势值策略（Trend Score）
- 交易方式：仅信号与通知，人工下单
- 核心触发：
  - 盘中每30分钟轮询（整点/半点）
  - 14:45最终可执行信号
  - 盘中若止损触发，允许提前ACTION-SELL

## 2. 已确认关键决策

- 数据源：`efinance` 优先，`akshare` 备选，接口层可插拔
- 复权口径：前复权
- 回测频率：日频
- 回测成交：当日收盘价（叠加滑点与手续费）
- 回测区间：优先2015-01-01至今，数据不足退化到2018-01-01至今
- 初始资金：默认20万（可调）
- 交易成本：
  - 手续费 `0.000085`（最低5元）
  - 滑点 `0.002`
- 组合与风控：
  - 每标的风险预算 `1.0%`
  - 硬止损ATR倍数：V1默认`1.5`（后续可按标的配置）
  - 资金超限：理论仓位等比缩放
  - 下单数量：100份整数倍向下取整
  - 允许少量现金残留（如0.1%）
- 交易限制：仅做多，严格T+1
- 运行与部署：
  - 本机长期运行
  - APScheduler
  - 先纯Python部署，后补Docker
- 存储：
  - 行情：Parquet
  - 配置/信号/回测/交易回写：JSON
  - 无数据库（V1）
- 通知：
  - 飞书机器人 + 邮件
  - 等级：INFO/WARN/ACTION/ERROR
  - 通知失败重试：2次，间隔5秒
  - 14:45拉取失败：重试3次，间隔20秒，仍失败当日不出ACTION
- 实盘一致性：
  - 系统ACTION仅建议
  - 你在收盘后至次日开盘前录入真实成交
  - 最终持仓以真实成交为准

## 3. 展示与审计

- 页面：
  - 系统概览
  - 策略配置
  - 回测中心
  - 日志审计
  - 成交回写
- 回测展示：
  - 指标：累计收益、年化、最大回撤、夏普、胜率、Calmar、Sortino、换手率、交易次数
  - 基准：4ETF等权
  - 年度收益表 + 月度收益热力图
  - 图表支持时间缩放联动
  - 图表标注买卖点
- 日志：
  - 永久保留
  - 每次轮询每个ETF记录完整中间计算过程
